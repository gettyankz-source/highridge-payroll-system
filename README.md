
# Highridge Construction Company Payroll Assignment

## Overview
This project contains Python and R programs developed to simulate a weekly payroll system for Highridge Construction Company.

The program:
- Dynamically generates a list of **400 workers**
- Uses a **for loop** to generate payment slips for each worker
- Applies **conditional logic** to assign employee levels
- Includes **exception handling**
- Provides both **Python and R implementations**

---

# Files Included

1. payroll.py  → Python implementation
2. payroll.R   → R implementation
3. README.md   → Instructions
4. highridge_payroll_assignment.zip → Submission package

---

# Python Instructions

## Requirements
Python 3.x

## Run the Program
Open a terminal and run:

python payroll.py

The program will:
1. Generate 400 workers dynamically
2. Assign salary and gender randomly
3. Evaluate employee level conditions
4. Print payment slips

---

# R Instructions

## Requirements
R installed on your system.

## Run the Program

Open R or RStudio and run:

source("payroll.R")

The program will generate workers and print payment slips.

---

# Employee Level Rules

Level A1:
Salary > 10,000 and < 20,000

Level A5-F:
Salary > 7,500 and < 30,000 AND employee is Female
